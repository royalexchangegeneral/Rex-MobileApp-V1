# Image Assets

Please place the following images in this folder:

1. **logo.png** - Rex Insurance logo (from 1.png)
2. **onboarding1.png** - World map with glowing network (from 4.png) - SCREEN 3
3. **onboarding2.png** - Car, plane, and animals scene (from 2.png) - SCREEN 1
4. **onboarding3.png** - Professional woman with phone (from 3.png) - SCREEN 2

## Image Requirements:
- Format: PNG with transparency (recommended)
- Resolution: @2x and @3x versions for better quality
- Naming convention: 
  - logo.png
  - onboarding1.png
  - onboarding2.png
  - onboarding3.png

## How to Add Images:
1. Save your images with the exact names above
2. Place them in this `assets/images/` folder
3. Run `flutter pub get` to ensure assets are recognized
4. Run the app with `flutter run`
